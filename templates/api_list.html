Authentication API

    Sign Up

    Request : POST

  • http://127.0.0.1:8000/register/

    • [email, password, confirm_password]

    Login

    Request : POST

  • http://127.0.0.1:8000/login/

    • [email, password]

    Get Menu, Category, Item, Modifier

    Request : GET

  • http://127.0.0.1:8000/menus/
  • http://127.0.0.1:8000/categories/
  • http://127.0.0.1:8000/items/
  • http://127.0.0.1:8000/modifiers/

    • Create Menu, Category, Item, Modifier

    Request : POST

  • http://127.0.0.1:8000/menus/create/

    • Basic Auth : owner or employee

    [name, description]

  • http://127.0.0.1:8000/categories/create/

    • Basic Auth : owner or employee

    [name,description, menu]

  • http://127.0.0.1:8000/items/create/

    • Basic Auth : owner or employee

    [name, description, price, available, category_id]

  • http://127.0.0.1:8000/modifiers/create/

    • Basic Auth : owner or employee

    [name, extra_price,item_id]
